# Электронный ассистент повара (Версия №2)

Современное Android-приложение для поиска, хранения и просмотра кулинарных рецептов, разработанное с использованием Jetpack Compose.

ElectroCook V2 является полной переработкой первой версии приложения ElectroCook и ориентировано на современные подходы Android-разработки.

## Основные возможности

- Просмотр каталога рецептов
- Поиск рецептов по названию
- Поиск рецептов по ингредиентам
- Фильтрация по калорийности
- Фильтрация по аллергенам
- Избранные рецепты
- Голосовой поиск
- Работа в оффлайн режиме
- Поддержка загружаемых пакетов рецептов
